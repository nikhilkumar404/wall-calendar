# Wall Calendar

A premium interactive wall calendar built with vanilla JavaScript.

## Features
- Monthly calendar grid with Monday-first layout
- Date range selection (click start → click end)
- Today highlight with amber ring
- Weekend styling in accent colour
- Quick selectors: This week / Next week / This month
- Notes per date range and per month (saved to localStorage)
- Month navigation with smooth animation
- Responsive — fits desktop viewport without scrolling
- Physical wall calendar aesthetic with hero image and wire

## Usage
Open index.html directly in your browser — no server needed.
